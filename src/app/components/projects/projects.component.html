Tax calculator Personal project

Recently I have been trying my hand in app development, I have created an app called the Tax Calculator. I decided to use Kotlin and Android Studio as it is based of Java so was able to quickly grasp the concepts. Once fully completed, it will be published to the Google Play store, available for download on Android devices. The purpose of the app is to calculate your tax and national insurance based on your annual income by using in app calculations with no external APIs. There are additional options such as student loans, pension contributions and overtime or bonus. An additional feature is a save and retrieve option, in which the user will be given a randomly generated alphanumeric code in which they are able to input this code and retrieve the details they had previously entered. I have achieved this by using the Firebase Realtime Database.
